# Inky Sitemap Pro 🚀

**Inky Sitemap Pro** is a lightweight, ultra-fast XML sitemap generator built for wordpress websites. Developed by the team behind [Inky Reads](https://inkyreads.com), this plugin bypasses the heavy database overhead of traditional SEO suites to provide instant indexing.

## 🌟 Why Inky Sitemap Pro?

Standard sitemap generators often crash or time out when dealing with 100,000+ nodes. This tool was engineered for the [Inky Reads Book Directory](https://inkyreads.com) to ensure that crawlers like Googlebot and Bingbot can efficiently discover deep-linked content without hitting server bottlenecks.

## 🛠 Features
- **Zero-Bloat Code**: No admin overhead or heavy UI.
- **Pretty URLs**: Automatically maps to `yoursite.com/inky-sitemap.xml`.
- **XSL Stylesheet**: Clean, readable XML output in the browser.

## 🚀 Installation
1. Download the `inky-sitemap-pro` folder.
2. Upload it to your `/wp-content/plugins/` directory.
3. Activate the plugin in the WordPress Admin.
4. Visit `yoursite.com/inky-sitemap.xml` to view your live sitemap.
